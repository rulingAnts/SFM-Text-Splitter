SFM Text Splitter

Split multi-text SFM (Toolbox/Shoebox) files into individual interlinear texts for FLEx import.

Download Latest (Windows Portable ZIP)

Fetches the latest release asset named "SFM Text Splitter.zip".

Purpose

FieldWorks Language Explorer (FLEx) cannot import multiple interlinear texts from a single file. This tool takes a single SFM format file containing multiple interlinear texts and splits it into separate files—one per interlinear text—ready for FLEx import.

Quick Start

  1. Download and unzip the Windows portable build.
  2. Run the app and choose Strict or Loose mode.
  3. Select your input SFM/text file.
  4. Select an EMPTY output folder.
  5. On success, the app opens the output folder and shows a completion dialog.
